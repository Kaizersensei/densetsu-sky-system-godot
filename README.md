# Densetsu Sky System

Public extraction of the Densetsu sky and weather system for Godot 4.6.

Included:
- `engine3d/sky` runtime scripts and scene
- `shaders/sky` shaders
- minimal cloud, moon, weather icon, and rain example assets
- standalone example scene: `engine3d/tests/Mesh Tester.tscn`

Notes:
- `Terrain3D` is not required.
- Weather particle scenes are optional. The example includes a simple rain particle scene.
- The system auto-creates internal helper nodes for cloud compositing and moon-surface rendering.
